# Azure Hybrid Network Lab

## Overview
This project demonstrates the design and implementation of a secure Azure network infrastructure, simulating a hybrid environment with on-premise connectivity concepts.

## Objectives
- Deploy and configure Azure Virtual Network (VNet)
- Implement subnetting and network segmentation
- Configure Network Security Groups (NSGs)
- Deploy and secure Virtual Machines
- Implement Private Endpoints for secure resource access
- Enable monitoring and diagnostics
